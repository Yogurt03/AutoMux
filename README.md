# AutoMux
本工具是为了懒人更便捷的配置Termux所写.

## 截图
<img src="Module/AutoMux.png">

### 依赖
• Python 3.x

#### 安装和使用方法
##### 懒人安装方法:
```
curl https://raw.githubusercontent.com/Bcap03/AutoMux/master/install.sh | bash
```
##### 手动安装方法:
```
git clone https://github.com/Bcap03/AutoMux.git
```
```
cd AutoMux
```
```
chmod +x install.sh
```
```
./install.sh
```
```
automux
```
